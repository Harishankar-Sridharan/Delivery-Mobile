# navigate_to

The **navigate_to** feature allows you to access and manage your saved navigation locations. Use this to quickly select destinations and streamline your route planning process.

### Getting Started

*   Ensure you have the Nomadia Delivery app installed.
*   Log in to your account.

1.  Open the application to the **main action screen**.

![Frame at 0:04](../images/navigate_to-call_timestamp_0_to_04.png "Main action screen – The application opens to the primary menu.")

2.  Tap the **navigate to** icon.

![Frame at 0:12](../images/navigate_to-call_timestamp_0_to_12.png "Navigation menu – The icon is selected to view saved locations.")

### Feature Overview

*   **Navigate To icon**: Opens the menu to view or add navigation points.

![Frame at 0:04](../images/navigate_to-call_timestamp_0_to_04.png "Main action screen – The navigate to icon is located in the center.")

*   **Where do you want to go page**: Displays all locations currently saved in the application.

![Frame at 0:12](../images/navigate_to-call_timestamp_0_to_12.png "Saved Locations list – A list of previously stored addresses is visible.")

*   **Plus button**: Initiates the process to save a new destination.

![Frame at 0:27](../images/navigate_to-call_timestamp_0_to_27.png "Where do you want to go page – The plus button is located at the bottom.")

*   **Name field**: Input area for the custom label of a location.
*   **Address field**: Input area for the physical destination address.
*   **Add**: Saves the entered information to your destination list.
*   **X button**: Returns you to the **main access page**.

### How To: Add a New Destination

1.  Tap the **navigate to** icon on the **main action screen**.

![Frame at 0:04](../images/navigate_to-call_timestamp_0_to_04.png "Main action screen – User taps the navigation icon.")

2.  Tap the **plus button** at the bottom of the **where do you want to go** page.

![Frame at 0:27](../images/navigate_to-call_timestamp_0_to_27.png "New Destination screen – User taps the plus button to add an entry.")

3.  Tap the **name field** and enter a location name.

4.  Tap the **address field** and enter the destination address.

![Flow 0:34 to 0:46](../images/navigate_to-call_timestamp_0_to_34_to_0_to_46.gif "New Destination form – User types the name and address into the respective fields.")

5.  Tap **add** to save the new location.

![Flow 1:04 to 1:09](../images/navigate_to-call_timestamp_1_to_04_to_1_to_09.gif "Save action – User taps the add button to confirm the new location.")

6.  Verify the new address appears in the list of available destinations.

7.  Tap the **X button** at the bottom to return to the **main access page**.

![Flow 1:15 to 1:21](../images/navigate_to-call_timestamp_1_to_15_to_1_to_21.gif "Exit – User taps the X button to close the menu.")

### Productivity Tips

- 💡 **Quick Navigation**: Save frequent destinations to the list for instant access during route execution.
