# agencies
# mobile

The agencies feature allows you to manage different operational hubs within Nomadia Delivery. It ensures your display shows data specific to a chosen location. Selecting an agency updates your dashboard with the relevant information for that site.

### Getting Started
*   Mobile device with Nomadia Delivery app installed.
*   Active user account with agency access permissions.

1. Open the **Main actions menu**.

![Frame at 0:03](../images/mobile-agencies_timestamp_0_to_03.png "Main actions menu – The initial navigation screen is shown.")

### Feature Overview
*   **Agencies**: This screen displays a list of all available locations for selection.

![Frame at 0:09](../images/mobile-agencies_timestamp_0_to_09.png "Agencies list – Multiple agency options are listed for the user to choose from.")

*   **Dashboard**: This interface displays the name of the agency you have currently selected.

![Frame at 0:14](../images/mobile-agencies_timestamp_0_to_14.png "Dashboard – The selected agency name is visible at the top of the interface.")

### How To: Select an Agency
1. Open the **Main actions menu**.

![Frame at 0:03](../images/mobile-agencies_timestamp_0_to_03.png "Main actions menu – The menu contains navigation options and configuration settings.")

2. Scroll down through the menu options.

![Flow 0:03 to 0:06](../images/mobile-agencies_timestamp_0_to_03_to_0_to_06.gif "Main actions menu – Navigating through the menu list to find the agencies section.")

3. Tap on **Agencies**.

![Frame at 0:06](../images/mobile-agencies_timestamp_0_to_06.png "Agencies button – The user selects the agencies option to open the selection screen.")

4. Select an agency name from the list.

![Flow 0:09 to 0:14](../images/mobile-agencies_timestamp_0_to_09_to_0_to_14.gif "Agency Selection – The user chooses a specific agency and the view returns to the dashboard.")

5. View the updated agency name on the **Dashboard**.

![Frame at 0:14](../images/mobile-agencies_timestamp_0_to_14.png "Dashboard – The selected agency is now confirmed and active on the main screen.")
