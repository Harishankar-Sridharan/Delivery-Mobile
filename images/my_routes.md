# my_routes

Access and manage your daily route schedules and delivery activities. Use this feature to view current assignments and plan for future delivery tasks efficiently,.

### Getting Started

*   Active Nomadia Delivery account.
*   Assigned routes for the selected date.

1.  Open the main actions menu.
2.  Tap **My Roots** to view assigned routes,.

![Frame at 0:07](../images/my_routes-call_timestamp_0_to_07.png "My Roots – Tapping the menu button to access assigned routes.")

### Feature Overview

*   **My Roots Page**: Displays the route schedule and associated delivery activities for the selected date,.

![Frame at 0:17](../images/my_routes-call_timestamp_0_to_17.png "My Roots Page – View of the daily schedule and delivery tasks.")

*   **Calendar Icon**: Opens a date picker in the upper right corner to change the viewed schedule,.

![Frame at 0:25](../images/my_routes-call_timestamp_0_to_25.png "Calendar Icon – Button to select a different date for the route schedule.")

*   **Back Arrow**: Navigation button in the upper left corner to return to the main actions menu,.

![Flow 0:55 to 1:00](../images/my_routes-call_timestamp_0_to_55_to_1_to_00.gif "Back Arrow – Sequence showing the return to the main menu.")

### How To: View and Change Route Schedules

1.  Tap **My Roots** from the main actions menu,.

![Frame at 0:07](../images/my_routes-call_timestamp_0_to_07.png "My Roots – Accessing the assigned routes list.")

2.  Tap the **calendar icon** in the upper right corner,.

![Frame at 0:25](../images/my_routes-call_timestamp_0_to_25.png "Calendar Icon – Opening the date selection tool.")

3.  Select a date, such as tomorrow's date, from the calendar,.

![Flow 0:26 to 0:31](../images/my_routes-call_timestamp_0_to_26_to_0_to_31.gif "Date Selection – Choosing a new date from the calendar popup.")

4.  Tap **OK** to confirm the selected date,.

![Flow 0:34 to 0:39](../images/my_routes-call_timestamp_0_to_34_to_0_to_39.gif "OK Button – Confirming the date to load the new schedule.")

5.  Review the refreshed route list for the selected day,.

![Flow 0:48 to 0:53](../images/my_routes-call_timestamp_0_to_48_to_0_to_53.gif "Refreshed List – The route schedule updates for the chosen date.")

6.  Tap the **back arrow** to exit the schedule view,.

![Flow 0:55 to 1:00](../images/my_routes-call_timestamp_0_to_55_to_1_to_00.gif "Back Arrow – Returning to the main action screen.")
