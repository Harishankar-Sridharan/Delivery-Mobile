# mobile

The **Logout** feature allows you to securely terminate your current application session. Dispatchers and planners use this to exit the mobile interface and protect account data. Completing this process ensures your specific login is closed properly.

### Getting Started

*   Active login session on the **Nomadia Delivery** mobile application.
*   Access to the **Main Actions** menu.

1.  Open the application to the home screen.

2.  Access the **Main Actions** list.

    ![Frame at 0:02](../images/mobile-logout_timestamp_0_to_02.png "Main Actions – The primary menu screen displaying available user features.")

### Feature Overview

*   **Logout**: A feature located at the bottom of the action list that ends the current session.

    ![Frame at 0:08](../images/mobile-logout_timestamp_0_to_08.png "Logout – The button used to exit the particular login.")

### How To: Log Out

1.  Open the **Main Actions** menu.

    ![Frame at 0:01](../images/mobile-logout_timestamp_0_to_01.png "Main Actions – Entering the main menu area.")

2.  Scroll down to the bottom of the list.

    ![Flow 0:02 to 0:05](../images/mobile-logout_timestamp_0_to_02_to_0_to_05.gif "Main Actions – Scrolling through the menu to reveal the hidden logout option.")

3.  Tap **Logout**.

    ![Frame at 0:07](../images/mobile-logout_timestamp_0_to_07.png "Logout – Selecting the logout option to finish the session.")

### Productivity Tips

- 💡 **Session Termination**: Tapping **Logout** will immediately exit you from that particular login session.
