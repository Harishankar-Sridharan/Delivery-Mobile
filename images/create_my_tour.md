# create_my_tour
# mobile

The Create My Tour feature allows dispatchers to manually build and prioritize specific delivery routes directly within the mobile application. Users can define departure times, scan package barcodes, and confirm route creation in real-time,. This ensures fleet activities are accurately captured and organized for daily operations.

### Getting Started

*   Access to the mobile application.
*   Scanning hardware or integrated camera for barcodes.

1. Open the **machines** section from the main actions menu,.

![Machines Menu – The main actions menu is visible with various options.](../images/mobile-create_my_tour_timestamp_0_to_07.png "Machines Menu – The main actions menu is visible with various options.")

2. Tap on **create my tool** to start the process,.

![Create My Tool – The button to initiate route creation is highlighted.](../images/mobile-create_my_tour_timestamp_0_to_13.png "Create My Tool – The button to initiate route creation is highlighted.")

### Feature Overview

*   **Road label**: Displays the unique identifier for the current route,.

![Road Label – The field showing the route identifier is displayed.](../images/mobile-create_my_tour_timestamp_0_to_19.png "Road Label – The field showing the route identifier is displayed.")

*   **Departure time**: Shows the scheduled start time for the delivery tour,.

![Departure Time – The field indicating when the route is set to begin.](../images/mobile-create_my_tour_timestamp_0_to_19.png "Departure Time – The field indicating when the route is set to begin.")

*   **Clock icon**: Opens the time selection interface,.

![Clock Icon – The icon used to set the departure time.](../images/mobile-create_my_tour_timestamp_0_to_27.png "Clock Icon – The icon used to set the departure time.")

*   **Calendar icon**: Opens the date picker for scheduling,.

![Calendar Icon – The icon used to select the date for the route.](../images/mobile-create_my_tour_timestamp_0_to_38.png "Calendar Icon – The icon used to select the date for the route.")

*   **order the route**: Toggle used to prioritize the specific route path,.

![Order Route – The toggle used to prioritize the route path.](../images/mobile-create_my_tour_timestamp_0_to_49.png "Order Route – The toggle used to prioritize the route path.")

*   **scan**: Activates the scanner to register package barcodes,.

![Scan Button – The interface for scanning package barcodes is shown.](../images/mobile-create_my_tour_timestamp_1_to_02.png "Scan Button – The interface for scanning package barcodes is shown.")

### How To: Create a Route

1. Tap on **create my tool** in the **machines** menu,.

![Create My Tool – The user starts the route creation process.](../images/mobile-create_my_tour_timestamp_0_to_13.png "Create My Tool – The user starts the route creation process.")

2. Tap the **Clock icon** next to the time,.

![Clock Icon – Selecting the clock icon to adjust time.](../images/mobile-create_my_tour_timestamp_0_to_19.png "Clock Icon – Selecting the clock icon to adjust time.")

3. Select the time and tap **OK**,.

![Flow 0:19 to 0:29](../images/mobile-create_my_tour_timestamp_0_to_19_to_0_to_29.gif "Time Selection – The user selects a time and confirms with the OK button.")

4. Tap the **Calendar icon** next to the date,.

![Calendar Icon – Opening the date selection menu.](../images/mobile-create_my_tour_timestamp_0_to_29.png "Calendar Icon – Opening the date selection menu.")

5. Select the date and tap **OK**,.

![Flow 0:29 to 0:41](../images/mobile-create_my_tour_timestamp_0_to_29_to_0_to_41.gif "Date Selection – The user selects a date and confirms with the OK button.")

6. Adjust the **order the route** toggle to prioritize your path,.

![Order Toggle – Adjusting the route priority.](../images/mobile-create_my_tour_timestamp_0_to_49.png "Order Toggle – Adjusting the route priority.")

7. Tap the **next icon** at the bottom of the screen,.

![Next Icon – Navigating to the scanning screen.](../images/mobile-create_my_tour_timestamp_0_to_54.png "Next Icon – Navigating to the scanning screen.")

8. Tap **scan** to scan the package barcode,.

![Scan Action – Activating the scanner for package barcodes.](../images/mobile-create_my_tour_timestamp_1_to_02.png "Scan Action – Activating the scanner for package barcodes.")

9. Tap **next** once scanning is complete,.

![Next After Scan – Moving forward after package scanning.](../images/mobile-create_my_tour_timestamp_1_to_10.png "Next After Scan – Moving forward after package scanning.")

10. Tap the **tick mark**,.

![Tick Mark – Submitting the route details for confirmation.](../images/mobile-create_my_tour_timestamp_1_to_14.png "Tick Mark – Submitting the route details for confirmation.")

11. Tap **confirm** on the confirmation pop-up,.

![Confirm Pop-up – Finalizing the route creation in the confirmation dialog.](../images/mobile-create_my_tour_timestamp_1_to_23.png "Confirm Pop-up – Finalizing the route creation in the confirmation dialog.")
