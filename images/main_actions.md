# main_actions
# mobile

The main actions menu allows you to manage your profile and view essential app information. Use this area to customize your identity and facilitate transfers between users. You will achieve a personalized profile and better coordination with other team members.

### Getting Started

*   Active account on the **Nomadia Delivery** mobile app.
*   Access to the mobile device camera for photo updates.

1. Open the app and start on the **dashboard** screen.
2. Tap the **three-line menu** in the top left corner.

![Flow 0:00 to 0:10](../images/mobile-main_actions_timestamp_0_to_00_to_0_to_10.gif "Main Actions Menu – Opening the sidebar menu from the dashboard.")

### Feature Overview

*   **Application Logo**: Displays your current profile picture in the sidebar.
![Frame at 0:32](../images/mobile-main_actions_timestamp_0_to_32.png "Application Logo – The primary profile image for the user.")

*   **Version Number**: Shows the current software version in the upper right corner.
![Frame at 0:40](../images/mobile-main_actions_timestamp_0_to_40.png "Version Number – Software version details for support and tracking.")

*   **QR Code Icon**: Provides access to your unique code for transfer operations.
![Frame at 0:40](../images/mobile-main_actions_timestamp_0_to_40.png "QR Code Icon – Entry point for generating a personal transfer code.")

### How To: Update Profile Image

1. Tap the **camera icon** displayed inside the logo area.

![Flow 0:42 to 0:47](../images/mobile-main_actions_timestamp_0_to_42_to_0_to_47.gif "Camera Icon – Initiating the profile image selection process.")

2. Select **choose an image** to pick a file from your gallery.

![Flow 0:51 to 1:00](../images/mobile-main_actions_timestamp_0_to_51_to_1_to_00.gif "Gallery Selection – Browsing and selecting an existing photo from the device.")

3. Select **take a picture** to capture a new photo with your camera.

![Flow 1:02 to 1:05](../images/mobile-main_actions_timestamp_1_to_02_to_1_to_05.gif "Camera Action – Using the device camera to snap a new profile photo.")

4. Tap **cancel** to stop the operation without making changes.

![Flow 1:11 to 1:15](../images/mobile-main_actions_timestamp_1_to_11_to_1_to_15.gif "Cancel Button – Reverting any unsaved changes to the profile image.")

### How To: Use QR Code for Transfers

1. Tap the **QR code icon** located next to the logo.

![Flow 1:20 to 1:25](../images/mobile-main_actions_timestamp_1_to_20_to_1_to_25.gif "QR Code Icon – Opening the personal transfer QR code screen.")

2. Display the **QR code** for another user to scan.

![Frame at 1:30](../images/mobile-main_actions_timestamp_1_to_30.png "QR Code Screen – The scanable code visible for peer-to-peer transfers.")

3. Tap the **back arrow** in the top left corner to exit.

![Flow 1:41 to 1:46](../images/mobile-main_actions_timestamp_1_to_41_to_1_to_46.gif "Back Arrow – Navigating away from the QR code screen back to the menu.")

### Productivity Tips

*   💡 **Personal Transfers**: Share your QR code with other users to record a personal transfer between team members.
*   ⚠️ **Cancel Changes**: Tap **cancel** if you select the wrong photo option to avoid updating your logo by mistake.
