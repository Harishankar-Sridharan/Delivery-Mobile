# assignthedepot

This feature allows you to assign a specific depot or building to a user profile through the mobile application,. It ensures that building assignments stay consistent by synchronizing data between the app and the back office,. Using this tool helps dispatchers maintain accurate location records for all field personnel.

### Getting Started

*   Active login credentials for the **Nomadia Delivery** mobile application,.
*   Access to the back office management portal to verify assignments,.

1. Open the mobile application and enter your login details,.

![Frame at 0:15](../images/assignthedepot-assignthedepot_timestamp_0_to_15.png "Dashboard – The main mobile screen displays the no deport option after login.")

### Feature Overview

*   **no deport**: Displays on the dashboard when a user has no building currently assigned,.

![Frame at 0:15](../images/assignthedepot-assignthedepot_timestamp_0_to_15.png "no deport – The status indicator showing no current building assignment.")

*   **Edit**: Opens the selection menu to choose a building from the assigned list,.

![Frame at 0:20](../images/assignthedepot-assignthedepot_timestamp_0_to_20.png "Edit button – The action link used to modify the depot assignment.")

*   **Manage Users**: The back office section where you can view and confirm user location assignments,.

![Frame at 0:55](../images/assignthedepot-assignthedepot_timestamp_0_to_55.png "Manage Users – The back office menu for overseeing user profiles.")

*   **Location**: The specific field in the back office that displays the synchronized building name,.

![Frame at 1:01](../images/assignthedepot-assignthedepot_timestamp_1_to_01.png "Location field – The field where the assigned building name is verified.")

### How To: Assign a Depot

1. Log in to the mobile application,.

![Frame at 0:15](../images/assignthedepot-assignthedepot_timestamp_0_to_15.png "Dashboard – The mobile home screen appearing after login.")

2. Locate the **no deport** option on the **Dashboard**,.

3. Tap the **Edit** option located directly under **no deport**,.

![Flow 0:15 to 0:20](../images/assignthedepot-assignthedepot_timestamp_0_to_15_to_0_to_20.gif "Edit action – Tapping the edit link to open building options.")

4. Review the list of buildings that have been assigned to your account,.

5. Select a building name, such as **George's B**,.

![Flow 0:20 to 0:33](../images/assignthedepot-assignthedepot_timestamp_0_to_20_to_0_to_33.gif "Building Selection – Choosing a specific building from the provided list.")

6. Verify that the selected building name now appears on the **Dashboard** page,.

![Frame at 0:42](../images/assignthedepot-assignthedepot_timestamp_0_to_42.png "Dashboard – The updated screen showing the newly assigned building.")

### How To: Verify Assignment in Back Office

1. Access the back office portal and click on **Manage Users**,.

![Flow 0:47 to 0:55](../images/assignthedepot-assignthedepot_timestamp_0_to_47_to_0_to_55.gif "Manage Users – Navigating to user settings in the back office.")

2. Click on the specific user you want to check,.

3. Locate the **Location** field to confirm it matches the building selected in the mobile app,.

![Frame at 1:01](../images/assignthedepot-assignthedepot_timestamp_1_to_01.png "Location field – The user profile displaying the synchronized building name.")
